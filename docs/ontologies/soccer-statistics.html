Description

Add-on ontology to IPTC Sport Schema, covering soccer statistics.

Properties

lineFormationRow p

IRI https://sportschema.org/ontologies/soccer/lineFormationRow
Description

Integer value ascending from closest to the goalkeeper. All players in the row closest to the goalkeeper have 1 as value, the next row 2, etc. Indicates row the player belonged to at the start of the game or when they entered the game.
Domain sport:Participation
Range xsd:integer

lineFormationPosition p

IRI https://sportschema.org/ontologies/soccer/lineFormationPosition
Description

Integer value ascending from left side of field. Starts at 1 for the first player in each line-formation. | Indicates line-position player had at the start of the game or when the player entered the game.
Domain sport:Participation
Range xsd:integer

lineFormation p

IRI https://sportschema.org/ontologies/soccer/lineFormation
Description

String representing the formation of the team on the field starting from the back of the field. This can be written as 4-4-2 where the hyphens separates the lines on the field from the defense at left to the forwards to the right. Depending on the formation of the team other than three digits can be used, for example 4-2-3-1.
Domain sport:Participation
Range xsd:integer
